# Retos diarios de Java 

Bienvenido al repositorio **“Retos diarios de Java”**, donde se almacenan ejercicios y desafíos de programación en Java 
proporcionados desde freeCodeCamp.

---

## Objetivo

El propósito de este repositorio es:

- Practicar Java de manera constante todos los días.
- Mejorar la lógica de programación y el dominio de estructuras de datos.
- Mantener constancia en proyectos diarios y el uso profesional de Git.
- Documentar cada reto de manera clara y ordenada para futuros revisores o empleadores.

---

## Organización del repositorio

Cada reto diario se encuentra en su propio paquete dentro de:
src/main/java/com/diego/retodiario/


**Convención de nombres de paquetes y clases:**

- Cada paquete se llama por la fecha del reto, por ejemplo:  
  `dia16Enero2026`
- Cada paquete contiene:
    - La clase Java con la solución, con un nombre descriptivo (`HipotenusaChecker.java`)
    - Un README.md específico del reto que explica el enunciado y la ejecución

**Ejemplo de estructura:**

    repositorio-retos/
    ├── README.md <-- Este README general
    └── src/
    └   ── main/
    └── java/
    └── com/
    └── diego/
    └── retodiario/
    ├── dia16Enero2026/
    │ ├── HipotenusaChecker.java
    │ └── README.md
    └── dia17Enero2026/
    ├── OtraClase.java
    └── README.md

(El ejemplo está basado en la estructura del primer día, la cual irá aumentando con el paso del tiempo.)
