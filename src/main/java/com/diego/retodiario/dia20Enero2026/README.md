# Reto 20/01/2026 – CompareEnergy

## Enunciado
Se recibe una cadena de caracteres con un mensaje.

En el mensaje se tienen que colocar todas las consonantes como mayúscula y las vocales como minúscula.

Los guiones se convierten en giones bajos.

---

## Entrada
Una única cadena de caracteres que contenga un mensaje

**Ejemplo de entrada:**
`Hello-wOrLd` 

---

## Salida
- La cadena de caracteres con el resultado tras la transformación del mensaje

**Ejemplo de salida:**

    Cadena introducida: Hello-wOrLd
    Resultado: HeLLo_WoRLD
