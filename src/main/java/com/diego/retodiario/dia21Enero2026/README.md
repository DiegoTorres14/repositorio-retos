# Reto 21/01/2026 – StringMarked

## Enunciado
Se recibe una cadena de marcado que puede contener parte de código en su interior.

El código se ve porque va entrecomillado con comillas simples --> 'ejemplo'.

---

## Entrada
Una única cadena de caracteres que contenga un mensaje

**Ejemplo de entrada:**
`Usa 'var' para definir una variable.`

---

## Salida
- La cadena de caracteres con el resultado tras la transformación del mensaje

**Ejemplo de salida:**

    Cadena introducida: Usa 'var' para definir una variable.
    Resultado: Usa <code>var</code> para definir una variable.
