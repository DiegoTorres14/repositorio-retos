# Reto 22/01/2026 – StringMarked

## Enunciado 
Se recibe un array de notas de exámenes. Las notas van de 0 a 100 y en función de la media se asigna un valor en forma de letra.

Las letras que se asignan son las siguientes:
    
    97-100	    "A+"
    93-96	    "A"
    90-92	    "A−"
    87-89	    "B+"
    83-86	    "B"
    80-82	    "B-"
    77-79	    "C+"
    73–76	    "C"
    70-72	    "C-"
    67-69	    "D+"
    63-66	    "D"
    60–62	    "D-"
    below 60    "F"
---

## Entrada
Se proporcionan al programa las notas sobre las que se calculará la media y en 
consecuencia se asigna la calificación.

**Ejemplo de entrada:**
`99` `98` `100`

---

## Salida
- El valor de la calificación.

**Ejemplo de salida:**

    Nota media --> 99
    Calificación --> A+
