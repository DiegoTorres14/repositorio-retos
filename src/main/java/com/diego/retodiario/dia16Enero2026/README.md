# Reto 16/01/2026 – HipotenusaChecker

## Enunciado
Se proporcionan dos números enteros positivos que representan las longitudes de los catetos de un triángulo rectángulo.  
El objetivo es determinar si la hipotenusa resultante es un número entero o decimal.

La longitud de la hipotenusa se calcula mediante el teorema de Pitágoras:

a² + b² = c²


donde `a` y `b` son los catetos y `c` es la hipotenusa.

---

## Entrada
Dos números enteros positivos pasados como argumentos de línea de comandos.

**Ejemplo de entrada:**
3 4

---

## Salida
- La hipotenusa calculada.
- Un mensaje indicando si la hipotenusa es un número entero (`true`) o decimal (`false`).

**Ejemplo de salida:**
Números:
3
4
La hipotenusa es número entero: true
