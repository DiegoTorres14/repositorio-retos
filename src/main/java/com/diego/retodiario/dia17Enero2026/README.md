# Reto 17/01/2026 – HipotenusaChecker

## Enunciado
Se proporciona la casilla en la que se encuentra el caballo en una partida de ajedrez.
El objetivo es conocer el número de movimientos que tiene disponibles dentro del tablero.

Para ello debemos conocer que el caballo se mueve saltando dos posiciones en vertical y una en horizontal o dos en 
horizontal y una en vertical

---

## Entrada
Una única cadena de caracteres cuya longitud es sí o sí de 2 caracteres.

El primer caracter será una letra desde la `A` a la `H`.

El segundo caracter será un número entre el `1` y el `8`.

**Ejemplo de entrada:**
`B5`

---

## Salida
- Número de posibles casillas a las que se puede mover el caballo en un mensaje proporcionado para el usuario.

**Ejemplo de salida:**
Para la posición `B5` existen `4` movimientos.
